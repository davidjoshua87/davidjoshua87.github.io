## 🚀 Getting Started

This is a **static website** deployed on GitHub Pages. No build process required!

### Local Development

1. Clone the repository:

   ```bash
   git clone https://github.com/davidjoshua87/davidjoshua87.github.io.git
   ```

2. Navigate to the project:

   ```bash
   cd davidjoshua87.github.io
   ```

3. Open `index.html` in your browser or use a local server:

   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .
   ```

### Deployment

- **GitHub Pages**: Automatically deployed on push to master branch
- **Netlify**: Drag and drop the folder
- **Vercel**: Import from GitHub

## 🔧 Customization

### 🎨 **Color Scheme**

Edit CSS variables in `styles.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... other colors */
}
```

### 📝 **Content**

Edit sections directly in `index.html`:

- Update personal information
- Add/remove projects
- Modify skills and experience
- Customize contact details

### 🖼️ **Images**

Replace image files:

- `profile.png` - Your professional photo
- `project-featured.png` - Featured project image
- Add new project images as needed
