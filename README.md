## ğŸš€ Getting Started

This is a **static website** deployed on GitHub Pages. No build process required!

### Local Development

1. Clone the repository:

   ```bash
   git clone https://github.com/davidjoshua87/davidjoshua87.github.io.git
   ```

2. Navigate to the project:

   ```bash
   cd davidjoshua87.github.io
   ```

   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js

   ```

- **Netlify**: Drag and drop the folder
